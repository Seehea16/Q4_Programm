--------------- SV Hausmannstätten - Trainerprogramm / HILFE ---------------
1.) Menüleiste
    1.1) Datei
        1.1.1) Speichern
            Speichert die aktuelle Tabelle mit Hilfe eines JFileChoosers ab.
        1.1.2) Laden
            Lädt ausgewählte Tabelle mit Hilfe eines JFileChoosers.
    1.2) Optionen
        1.2.1) README-Datei anzeigen
            Zeigt eine Hilfe-Datei zum besseren Verständnis des Programms an.
        1.2.2) Grafischen "Stammbaum" anzeigen
            Zeichnet einen grafischen Stammbaum, um eine bessere Übersicht
            über die Vereinshierarchie zu erhalten.
        1.2.3) Programm beenden
            Beendet das Programm.
2.) Filter
    1.1) RadioButtons
        Mithilfe der RadioButtons Alle, Vorstand, Trainer und Spieler kann
        nach der jeweils ausgewählten Art gefiltert werden.
    1.2) CheckBox Mannschaft
        Falls der RadioButton Spieler ausgewählt ist, erscheint eine Check-
        Box Mannschaft, mit der man noch nach Mannschaft filtern kann.
    1.3) ComboBox Trainingsdatum
        Im Falle, dass auch die Checkbox Mannschaft selektiert wurde, kann
        nun noch nach einem bestimmten Training gefiltert werden, um so
        die An- und Abwesenheiten eintragen zu können (Falls man Trainer ist).
3.) Table
    3.1) Allgemein
        In der Tabelle werden die verschiedensten Daten zu Vereinspersonen
        angezeigt, um eine bessere Übersicht über die Mitglieder des Vereins
        zu erhalten.
    3.2) Rechtsklick
        Per Rechtsklick können entweder ausgewählte Personen gelöscht, oder
        mithilfe eines Dialogs neue Personen hinzugefügt werden.
    3.3) Ändern der Daten
        Mit einem doppelten Rechtsklick lassen sich Daten aus der Tabelle
        einfach manipulieren.
